# sumRom.py
Shade3Dのプラグイン。選択した稜線の長さの合計を求める。

## 使い方
1. Shade3Dの『メインメニュー』の『表示』より『スクリプト』をクリックします。
2. 表示されたウインドウにソースコードをコピーもしくは『読込』ボタンをクリックしファイルを指定します。
3. 『実行』ボタンをクリックすると実行されます。
